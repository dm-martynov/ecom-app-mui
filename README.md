# Ecommerce App

### Built with React and Express.js

## Key Features

- Sign-Up/Sign-In with validation and api error popups
- Fully Responsive
- 2 types of sorting
- Currency changing
- Adding and removing items from cart
- Checkout using stripe API
- Infinite Scroll
